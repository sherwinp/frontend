Fill-in Form 8809

If you need to file for an extension of time for forms W-2, W-2G, 1042-S, 1098 series, 1099 series, 5498 series or 8027, you can now do it online via the FIRE system in lieu of filing out the paper Form 8809 and mailing it to us. From the main menu, click on 'Extension of Time Request', then click on 'Fill-in Extension Form' and follow the prompts.

If you are requesting an extension for more than 10 payers, you must file electronically. See Publication 1220, Part D or Tax Topic 803 at www.irs.gov for more details.

PLEASE NOTE: This is not for recipient copies and is only for the forms listed above.
Combined Federal State Filing Program Enhances Validation Process

The Combined Federal State Filing Program was established to simplify reporting of Information Returns to participating states. Due to increased concerns over proper disclosure of information, the Information Reporting Program is enhancing the validation process of combined federal/state data.

To ensure that your files are accurate, we encourage all participants to send a test file annually prior to submitting your original information returns. IRS will accept test files electronically through the FIRE System from November 1 through February 15.

For additional information on the Combined Federal/State Filing Program, submitting test files, or becoming a new participant in the program refer to Publication 1220, Specifications for Filing Forms 1098, 1099, 5498 and W-2G Electronically. This publication can be located on the IRS website at www.irs.gov

File Size

This system is only capable of accepting a file with up to approximately 5 million records (at 750 bytes per record). We encourage you to compress the file using PKZIP, or WINZIP compression methods. At 92% compression, this equates to a file size of approximately 330 MB. If you have files larger than this, please call us at 1-866-455-7438 for specific instructions.

REPLACEMENT or CORRECTION FILES If you have sent an original or correction file and FILE STATUS indicates the file is bad, you must send the corrected file as a replacement file, not a correction or original. If you are still confused, please call us toll free at 1-866-455-7438.
